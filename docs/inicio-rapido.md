# Inicio Rápido

1. Abra QGIS y vaya a **Complementos → Censo Argentino**
2. En la pestaña **Explorar**, configure:
    - **Año**: 2022
    - **Nivel geográfico**: Radio Censal (más detalle) o Provincia (más rápido)
    - **Tipo de entidad**: Hogar, Persona o Vivienda
3. Busque una variable (ej: escriba "edad" en el buscador)
4. Marque las variables que desea cargar
5. Haga clic en **Cargar Capa**

Los datos aparecen como una capa de polígonos con las variables seleccionadas como columnas de atributos.

Consulte la [Guía de Usuario](guia-usuario.md) para opciones avanzadas o el [Modo SQL](sql.md) para consultas personalizadas.
